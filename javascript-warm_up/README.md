# JavaScript Warm Up

This directory contains introductory JavaScript scripts and exercises for the Holberton School curriculum. Each file demonstrates a basic concept or syntax in JavaScript, such as variables, printing to the console, and more.

## Contents
- 0-javascript_is_amazing.js: Prints "JavaScript is amazing" to the console.

## Usage
Make sure you have Node.js installed. To run a script:

```bash
./0-javascript_is_amazing.js
```
or
```bash
node 0-javascript_is_amazing.js
```

## Requirements
- Node.js
- semistandard (for linting)
