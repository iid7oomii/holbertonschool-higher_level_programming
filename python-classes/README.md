Project structure explanation

Each top-level folder groups exercises around a particular Python topic (for example, "python-classes" contains class- and OOP-related exercises). Files are individual exercises or example scripts that can be executed independently.

Setup and installation

1. Clone the repository:

   git clone https://github.com/iid7oomii/holbertonschool-higher_level_programming.git

2. Enter the project directory:

   cd holbertonschool-higher_level_programming

3. Ensure you have Python 3 installed (Python 3.8+ recommended):

   python3 --version

How to run the project

- Run individual Python exercises directly with Python 3. For example, to run the example script at the repository root:

  python3 0-square_matrix_simple.py

- Exercises inside folders can be run the same way; change into a directory or reference the file path.

Configuration or environment variables

- This repository does not require any configuration files or environment variables to run the example scripts. If an exercise requires additional setup, it will be noted in that exercise's file or folder.

Example usage

- Execute a script from the repository root:

  python3 0-square_matrix_simple.py

- Run a class exercise inside the python-classes folder (replace <exercise>.py with an actual filename if present):

  python3 python-classes/<exercise>.py
